# 05 · Napoje i nawodnienie

## 1. Zapotrzebowanie na płyny

> **Dieta w czasie chemioterapii musi zawierać dużo wody — ok. 2,5 l / dobę.**
>
> Najczęstszy błąd, który przyczynia się do nasilenia skutków ubocznych, to **brak odpowiedniej ilości wody w organizmie.**

### Cele właściwego nawodnienia
- wspomaga eliminację metabolitów leków,
- łagodzi nudności i zaparcia,
- zapobiega odwodnieniu po wymiotach / biegunce,
- utrzymuje stabilną energię w ciągu dnia.

- 🚫 Cukier biały w nadmiarze
- 🚫 Słodziki sztuczne nieprzebadane

## 5. Praktyczna reguła dla modelu

```python
def rekomendacja_plynow(pacjent, objawy):
    plan = {
        "min_ml_na_dobe": 2500,
        "zalecane": ["woda_lekko_gazowana_mineralna"],
        "unikac": ["napoje_gazowane_kolorowe", "alkohol", "soki_cytrusowe_grejpfrut"],
    }
    if "biegunka" in objawy:
        plan["zalecane"] += ["herbata_z_kopru_wloskiego", "kakao_na_wodzie", "kleik_ryzowy"]
        plan["unikac"]   += ["mleko_swieze", "kawa_naturalna", "soki", "alkohol"]
    if "zaparcia" in objawy:
        plan["zalecane"] += ["woda_z_miodem_przed_sniadaniem", "kompot_z_suszonych_owocow_jablek", "kawa_naturalna"]
    if "nudnosci_wymioty" in objawy:
        plan["zalecane"] += ["herbata_z_imbirem", "herbata_mietowa", "kostki_lodu"]
        plan["unikac"]   += ["napoje_zbyt_gorace", "mocno_pachnace"]
    if "zapalenie_jamy_ustnej" in objawy:
        plan["unikac"]   += ["napoje_zbyt_gorace", "soki_cytrusowe", "alkohol", "gazowane"]
    if "wzdecia" in objawy:
        plan["unikac"]   += ["sok_z_buraka_nadmiar", "gazowane"]
        plan["zalecane"] += ["herbata_mietowa", "herbata_z_kopru_wloskiego"]
    return plan
```
